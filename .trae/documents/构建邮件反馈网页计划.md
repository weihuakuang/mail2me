# 使用 Next.js 构建反馈网页计划

## 目标
使用 Next.js 创建一个反馈网页，利用 Vercel Serverless Functions 处理邮件发送逻辑，方便后续一键部署到 Vercel。

## 技术栈
- **框架**: Next.js (React)
- **样式**: Tailwind CSS (快速构建美观界面)
- **邮件服务**: Nodemailer
- **部署目标**: Vercel

## 实施步骤

### 1. 项目初始化
- 使用 `create-next-app` 在当前目录初始化项目。
- 配置：TypeScript, Tailwind CSS, App Router。
- 安装 `nodemailer` 依赖。

### 2. 后端开发 (API Route)
- 创建 API 路由 `app/api/send-email/route.ts`。
- 在此路由中配置 Nodemailer，使用 QQ 邮箱 SMTP 服务 (`smtp.qq.com`)。
- 实现 POST 方法处理逻辑：接收前端 JSON 数据 -> 验证 -> 发送邮件到 `2446563904@qq.com`。

### 3. 前端开发 (`app/page.tsx`)
- 设计一个现代化的反馈表单（包含姓名、邮箱、反馈内容）。
- 使用 React `useState` 管理表单输入和提交状态（加载中、成功、失败）。
- 使用 `fetch` 调用 `/api/send-email` 接口。
- 添加输入验证和友好的用户提示。

### 4. 环境变量配置
- 创建 `.env.local` 文件。
- 定义环境变量 `EMAIL_USER` (你的邮箱) 和 `EMAIL_PASS` (SMTP 授权码)。
- **注意**：部署到 Vercel 时，需要在 Vercel 项目设置中添加这些环境变量。

## 准备工作
- 请准备好 QQ 邮箱的 **SMTP 授权码**。
